# Memory Ledger — потери памяти Нэйры (негативный канон)

Что Нэйра **больше не помнит**. Несущая балка сюжета: каждая запись необратима.

## Правила ведения

- Писатель не имеет права вернуть Нэйре забытое; критика, ревизия и `arc_sync` проверяют каждую главу против этого списка.
- Удаление или ослабление записи — реткон: остановись и спроси пользователя.
- Частичное «возвращение» возможно только через людей и заново прожитое (см. кандидат-финал в `outline.md`), фиксируется **новой записью** с обоснованием, а не удалением потери.
- После стабилизации главы с потерей: перенести её из «запланированных» в «канонические» с номером главы и тем, чем оплачена.

## Запланированные потери (по outline; становятся каноном после своих глав)

- Часть I, учебное Якорение: имя детской подруги.
- Зона 1 (Лес страхов): лицо отца / ранний домашний якорь.
- Зона 2 (Боковая равнина): «зачем идёт» — помнит цель-город, забывает, что там семья.
- Зона 3 (Помнящие руины): кто есть кто из спутников; держится на внешней опоре (Иш/Кейран).
- Финал (Стены Сэл'Морна): вся память о себе — чистый лист.
