# Mistakes — «Картографы невозможного»

Короткие переиспользуемые правила письма для этой книги (не жалобы по конкретной главе). Пополнять по ходу критики/ревизий.

- Магия — переговоры, не приказ: показывать цену каждого Якорения, не делать силу бесплатной.
- Потерю памяти показывать конкретной деталью (пропавшее имя, чужое лицо), а не словом «забыла».
- Не описывать разные Якорения одинаково; беречь эскалацию цены к финалу.
- Не «возвращать» забытое случайно — сверяться с реестром потерь.
- Не выводить прочие дисциплины Морвейна на первый план; фокус — Якорение.
- Избегать «избранной», мгновенного мастерства и deus ex machina у стен города.
