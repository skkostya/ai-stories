# Events Memory

## Confirmed Canon Events

- Звездный дождь стал отправной точкой всех ключевых событий проекта.
- Канонические события по главам добавляются сюда после утвержденных текстов.
- `## Chapter 02`
- Нора в течение нескольких дней после звездного дождя фиксирует повторяющийся аномальный паттерн в видеофайле с ночи падения.
- Школьная техника и общественные системы начинают давать локальные сбои в присутствии Норы, но для окружающих это выглядит как обычные поломки.
- В библиотеке Нора сохраняет фрагмент сигнала с автоматическим именем `sig_18-07_red`.
- Дома аномалия распространяется на несколько устройств сразу и впервые адресно откликается на Нору через шум, текст и ритм.
